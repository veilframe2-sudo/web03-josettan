# Josettan's · Cinema Company Restaurant & Caterers

A single-page site for a cinema-themed multi-cuisine restaurant & caterer in
Viyyur, Thrissur. Built as production vanilla HTML/CSS/JS from a Claude Design
prototype.

**Live site:** https://veilframe2-sudo.github.io/web03-josettan/

## Structure
- `index.html` — all sections, content baked in
- `styles.css` — full stylesheet
- `app.js` — interactivity (nav scroll-spy, catering wizard, menu tabs,
  gallery lightbox, seat-map reservation, toasts)
- `images/` — dish, interior, and director photography

No build step — open `index.html` or serve the folder statically.
